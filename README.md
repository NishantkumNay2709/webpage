# 🎨 CSS Basics

A beginner's guide to CSS, including linking CSS to HTML, selectors, and colors.

## 📂 Project Structure

```
project/
│── index.html
└── style.css


## 🔗 Linking CSS to HTML

```html
<head>
    <link rel="stylesheet" href="style.css">
</head>
```

---

## 📝 CSS Syntax

```css
selector {
    property: value;
}
```

Example:

```css
h1 {
    color: blue;
}
```

---



---

## 🎨 CSS Colors

```css
/* Text Color */
p {
    color: blue;
}

/* Background Color */
body {
    background-color: lightgray;
}
```

Different color formats:

```css
color: red;
color: #ff0000;
color: rgb(255,0,0);
color: hsl(0,100%,50%);
```
